# TaskMaster iOS App

A task management iOS application built using SwiftUI and MVVM architecture. This app helps users efficiently manage their daily tasks with a clean and user-friendly interface.

---

## 🚀 Features
- Add, update, and delete tasks
- Mark tasks as completed
- Task priority support
- Persistent data storage using UserDefaults
- Simple and responsive UI

---

## 🛠️ Tech Stack
- Swift
- SwiftUI
- MVVM Architecture
- UserDefaults

---

## 📱 Description
TaskMaster is a simple yet effective task management application designed to demonstrate core iOS development concepts such as state management, MVVM architecture, and local data persistence. The app focuses on providing a smooth user experience with minimalistic design.

---

- Push notifications for tasks

- Task categorization
